## Instalation local

Cloner le projet

```bash
  git clone https://github.com/gabrielvoissiere/website.git
```

Aller dans le projet

```bash
  cd website
```

Instaler les dependances

```bash
  npm install
```

Lancer le serveur

```bash
  npm run serve
```
